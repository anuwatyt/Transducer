## 2. Sensor Fundamentals

เซ็นเซอร์คืออุปกรณ์ที่ใช้ตรวจจับการเปลี่ยนแปลงของสิ่งแวดล้อมทางกายภาพ แล้วแปลงการเปลี่ยนแปลงนั้นให้กลายเป็นสัญญาณไฟฟ้าที่ระบบควบคุมอ่านได้ เช่น ไมโครคอนโทรลเลอร์, PLC หรือคอมพิวเตอร์ฝังตัว

ตัวอย่างสิ่งที่เซ็นเซอร์มักตรวจจับได้:

- แสง
- อุณหภูมิ
- ความชื้น
- ระยะทาง
- การเคลื่อนไหว
- แรงดัน
- แม่เหล็ก
- การสั่นสะเทือน

### 2.1 หลักการทำงานพื้นฐาน

โดยทั่วไปเซ็นเซอร์จะมี 3 ส่วนสำคัญ:

1. **Sensing element**  
   ส่วนรับรู้ค่าทางกายภาพ เช่น แผ่น reed ใน KY-021
2. **Signal conditioning**  
   ส่วนปรับสภาพสัญญาณ เช่น กรองสัญญาณ, ขยายสัญญาณ, หรือเปรียบเทียบค่า
3. **Output interface**  
   ส่วนส่งออกข้อมูลไปยังไมโครคอนโทรลเลอร์แบบ Analog หรือ Digital

### 2.2 คำสำคัญที่ควรรู้

| คำศัพท์ | ความหมาย |
|---|---|
| Range | ช่วงค่าที่เซ็นเซอร์วัดได้ |
| Sensitivity | ความไวต่อการเปลี่ยนแปลงของสัญญาณ |
| Accuracy | ความใกล้เคียงกับค่าจริง |
| Resolution | ความละเอียดของค่าที่อ่านได้ |
| Response time | เวลาที่เซ็นเซอร์ตอบสนองต่อการเปลี่ยนแปลง |
| Hysteresis | ความแตกต่างของจุดทำงานตอนเพิ่มและลดสัญญาณ |
| Calibration | การปรับเทียบให้ผลวัดแม่นยำขึ้น |

### 2.3 เซ็นเซอร์แบบ Analog และ Digital

- **Analog sensor** จะให้ค่าเป็นช่วงต่อเนื่อง เช่น 0-1023 หรือแรงดันไฟฟ้า
- **Digital sensor** จะให้ผลเป็นสถานะ เช่น `HIGH/LOW` หรือ `1/0`

KY-021 เป็น **เซ็นเซอร์แบบดิจิทัล** เพราะเอาต์พุตหลักเป็นสถานะเปิด/ปิดเมื่อมีสนามแม่เหล็กเข้าใกล้

### 2.4 ข้อควรคำนึงในการใช้งานเซ็นเซอร์

- อ่านช่วงแรงดันไฟเลี้ยงให้ตรงกับบอร์ด
- ดูว่าเอาต์พุตเป็น Analog หรือ Digital
- ตรวจสอบการต่อขาให้ถูกต้อง
- พิจารณา pull-up / pull-down resistor
- ทดสอบอาการสัญญาณเด้ง หรือ noise ถ้าเป็นสัญญาณสวิตช์

## 3. ข้อมูล KY-021 (Datasheet Summary)

KY-021 คือโมดูล **Magnetic Reed Switch** หรือ **Mini Reed Switch Module** ใช้ตรวจจับสนามแม่เหล็กจากสวิตช์แบบ reed switch ภายในหลอดแก้วปิดสนิท

### 3.1 หลักการทำงานของ reed switch

ภายในหลอดแก้วจะมีแผ่นโลหะบาง 2 แผ่น เมื่อไม่มีแม่เหล็ก แผ่นโลหะจะแยกจากกันและวงจรจะยังไม่ปิด

เมื่อมีแม่เหล็กเข้าใกล้:

- สนามแม่เหล็กจะเหนี่ยวนำให้แผ่นโลหะเป็นแม่เหล็กชั่วคราว
- แผ่นโลหะจะดูดเข้าหากัน
- วงจรจึงปิดและเปลี่ยนสถานะเอาต์พุต

เมื่อเอาแม่เหล็กออก:

- แผ่นโลหะจะคืนรูปเดิม
- วงจรเปิดอีกครั้ง

### 3.2 สเปกหลัก

> หมายเหตุ: สเปกบางรายการอาจต่างกันเล็กน้อยตามผู้ผลิตหรือล็อตสินค้า แต่ช่วงค่าด้านล่างเป็นค่าที่พบได้บ่อยจากหน้าสินค้าอ้างอิง

| รายการ | ค่าโดยประมาณ |
|---|---|
| ชื่อโมดูล | KY-021 Reed Switch Module |
| ประเภทเซ็นเซอร์ | Magnetic Reed Switch |
| ประเภทเอาต์พุต | Digital |
| แรงดันใช้งาน | 3.3V - 5V |
| สีบอร์ด | ดำ |
| ขนาดบอร์ด | ประมาณ 2.4 x 1.5 x 0.5 ซม. |
| จำนวนขา | 3 ขา |
| การใช้งานหลัก | ตรวจจับประตู, หน้าต่าง, ตำแหน่ง, นับรอบ, งาน alarm |

### 3.3 ขาเชื่อมต่อ

| ขา | ชื่อ | หน้าที่ |
|---|---|---|
| 1 | VCC | ไฟเลี้ยง 3.3V ถึง 5V |
| 2 | GND | กราวด์ |
| 3 | DO | Digital Output |

### 3.4 ลักษณะสัญญาณเอาต์พุต

เมื่อใช้ร่วมกับไมโครคอนโทรลเลอร์:

- ไม่มีแม่เหล็กใกล้เซ็นเซอร์ -> ค่าจะอยู่ที่สถานะหนึ่ง
- มีแม่เหล็กใกล้เซ็นเซอร์ -> ค่าจะเปลี่ยนสถานะทันที

ในทางปฏิบัติ โหมดอ่านค่าที่ใช้บ่อยคือ `INPUT_PULLUP` แล้วอ่านสถานะ `LOW` เมื่อมีการปิดวงจร

## 5. ตัวอย่างโค้ด Arduino

```cpp
const int reedPin = 2;
const int ledPin = 13;

void setup() {
  pinMode(reedPin, INPUT_PULLUP);
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int state = digitalRead(reedPin);

  if (state == LOW) {
    digitalWrite(ledPin, HIGH);
    Serial.println("Magnet detected");
  } else {
    digitalWrite(ledPin, LOW);
    Serial.println("No magnet");
  }

  delay(200);
}
```

## 6. การใช้งานที่เหมาะสม

- ตรวจจับการเปิด-ปิดประตูหรือหน้าต่าง
- ระบบ alarm หรือ security
- นับจำนวนรอบเมื่อมีแม่เหล็กผ่านตำแหน่งเดิม
- ตรวจจับตำแหน่งชิ้นส่วนในงานอัตโนมัติ
- โปรเจกต์ Arduino สำหรับผู้เริ่มต้น

## 7. ข้อควรระวัง

- อย่าให้หลอดแก้วของ reed switch กระแทกแรง
- ใช้แม่เหล็กที่มีแรงพอเหมาะกับระยะตรวจจับ
- ถ้าสัญญาณแกว่ง ควรทำ debounce ในซอฟต์แวร์
- ตรวจสอบแรงดันไฟให้ตรงกับโมดูลก่อนต่อใช้งาน
- โมดูลแต่ละยี่ห้ออาจจัดวางขาและอุปกรณ์บนบอร์ดไม่เหมือนกันทั้งหมด

## 8. สรุป

KY-021 เป็นเซ็นเซอร์แบบดิจิทัลที่เข้าใจง่าย ใช้หลักการของ reed switch ในการตรวจจับสนามแม่เหล็ก จึงเหมาะกับงานตรวจจับสถานะเปิด-ปิด การนับตำแหน่ง และงานพื้นฐานด้านไมโครคอนโทรลเลอร์

หากต้องการใช้งานกับ Arduino UNO ก็สามารถต่อวงจรได้ง่ายมาก และเขียนโค้ดอ่านค่าได้ไม่ซับซ้อน
